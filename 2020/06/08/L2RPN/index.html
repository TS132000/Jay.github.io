Action

定义两个动作类型：Change vs Set

Change：改变

  • 连接未连接的电线，重连已连接的电线
  • 母线1的object连到母线2，母线2的object连到母线1

Set：设置

  • 不论电线什么状态，连接电线or断掉电线
  • 不论object在哪个母线，将其设置为连在母线X上。

    模糊动作类型

    不能被正确or唯一解释的动作，这类动作如果做了将被无视，与没做相同。此类动作有很多，具体见官方文档_check_for_ambiguity.

Action Helper / action space

an Action can modify production, loads, topology,etc
